Selected findings · synthetic data

Examples of how Blackridge qualifies a conclusion.

Observed behavior is kept separate from modeled opportunity so buyers can distinguish recorded evidence from changes that still need validation.

Observed

Retry amplification

Repeated attempts share a workflow and failure window. The example attributes $118.40 of observed waste to retry behavior and recommends reviewing retry bounds and failure classification.

Modeled

Fallback cost variance

The fallback path moved failed calls to a higher-cost model. The $61–$84 range remains modeled until engineering validates acceptable fallback behavior.

Inferred

Duplicate-work opportunity

Repeated request evidence suggests a caching or de-duplication candidate. Confidence remains limited where fingerprint coverage is incomplete.
