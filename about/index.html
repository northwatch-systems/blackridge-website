Company

Northwatch Systems builds Blackridge.

Blackridge is a Token Forensics Engine / Gateway for enterprise inference spend. It sits between AI applications and model providers to make inference spend attributable, explainable, and controllable.

Mission

Make token spend controllable.

AI costs increasingly come from runtime behavior: retries, fallbacks, agent branches, duplicate requests, model defaults, missing attribution, and data-quality gaps. Northwatch Systems builds Blackridge so engineering and finance can inspect evidence and decide which runtime controls are safe to deploy.

Infrastructure-native

Blackridge is built for engineering teams working with runtime traffic, logs, traces, provider events, metadata, and workflow evidence.
